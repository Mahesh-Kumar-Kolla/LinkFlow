# LinkFlow

A modern URL redirect chain analyzer built with Next.js. Uncover where shortened URLs really lead - perfect for security checks, debugging, and transparency.

## Features

- **Complete Redirect Chain** - See every hop from start to final destination
- **Response Times** - Measure latency at each redirect step (in milliseconds)
- **Status Codes** - Color-coded HTTP status badges (2xx, 3xx, 4xx, 5xx)
- **One-Click Copy** - Easily copy any URL in the chain
- **Mobile Responsive** - Works beautifully on all devices
- **SSRF Protection** - Blocks requests to localhost and private IPs
- **Modern UI** - Glassmorphism design with smooth animations

## Tech Stack

- **Framework:** Next.js 16.1
- **Language:** TypeScript
- **UI Library:** React 19
- **Styling:** Tailwind CSS 4
- **Fonts:** Geist (Google Fonts)

## Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/Mahesh-Kumar-Kolla/linkflow.git
   cd linkflow
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   ```

4. **Open in browser**
   
   Visit [http://localhost:3000](http://localhost:3000)
